# 🗳️ Tamil Nadu Election 2026 Analysis Dashboard  

🔗 **Live Demo:** https://krithika-10.github.io/Tamil-Nadu-Election-2026-Analysis/

---

## 📌 Overview  
This project is an interactive dashboard focused on analyzing **Tamil Nadu Election 2026** data. It presents party performance, vote share trends, and comparisons in a simple and visually clear way, making it easy for both technical and non-technical users to understand.

---

## 🎯 Objectives  
- Analyze party-wise performance and vote share  
- Identify trends and comparisons between parties  
- Present insights through an interactive dashboard  

---

## 🛠️ Technologies Used  
- HTML  
- CSS  
- JavaScript  
- Chart.js (visualizations)  

---

## 🤖 AI-Assisted Development  
- Designed the overall dashboard structure and analysis approach  
- Created structured prompts to guide AI (Claude) in generating code  
- Clearly specified charts, layout, and data flow  
- Reviewed and refined outputs to improve accuracy and usability  

---

## 📊 Key Features  
- Interactive charts showing vote share and party comparisons  
- Clear visualization of trends and performance  
- User-friendly dashboard with simple insight sections
